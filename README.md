# 📊 Excel Sales Performance Dashboard

## Project Overview
An interactive Excel dashboard built using Pivot Tables, Pivot Charts, Slicers, and KPI Cards.

## Dashboard Preview

![Dashboard](Images/dashboard.png)

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Sales

## Features

- Sales by Region
- Monthly Sales Trend
- Profit by Category
- Payment Mode Analysis
- Salesperson Performance
- Interactive Filters

## Skills Demonstrated

- Data Cleaning
- Data Analysis
- Dashboard Design
- Excel Automation
